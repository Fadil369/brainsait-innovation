## 📚 Course Units Overview

### Unit 1: Introduction to Innovation (45 min)
- Welcome & Course Introduction
- What is the Toolkit?
- Innovation in UN Context
- Navigating the Toolkit
- **Assessment:** 5 MCQ questions

### Unit 2: S.P.A.C.E. Framework (60 min)
- What is S.P.A.C.E. and Why it Matters
- Strategy Module
- Partnerships Module
- Architecture Module
- Culture Module
- Evaluation Module
- **Assessment:** 8 questions (MCQ + Short Essay)

### Unit 3: Innovation Diagnostic (90 min)
- What is the Innovation Diagnostic?
- Strategy Module Questions
- Partnership Module Questions
- Architecture Module Questions
- Culture Module Questions
- Evaluation Module Questions
- Taking the Diagnostic
- **Activity:** Complete diagnostic and record results

### Unit 4: Innovation Profiles (45 min)
- Understanding Profiles
- Motivator Profile
- Strategist Profile
- Collaborator Profile
- Implementor Profile
- Early-Stage Innovator Profile
- Trailblazer Profile
- **Activity:** Identify and share your profile

### Unit 5: Strategy Tools (75 min)
- Headlines of the Future
- Scenario Blueprint
- Ecosystem Analysis
- Portfolio Strategy
- Innovation Planner
- **Project:** Apply one strategy tool

### Unit 6: Partnership Tools (60 min)
- Define a Value Proposition
- Find Different Partners
- Prepare to Partner
- Prioritize and Select Partners
- **Project:** Identify potential partner and use value proposition tool

### Unit 7: Architecture Tools (75 min)
- Scan the Horizon
- User-Centered Design
- From Pilot to Scale
- Operating Model
- **Project:** Design a pilot using "From Pilot to Scale"

### Unit 8: Culture Tools (60 min)
- Embrace Failures
- Create Incentives and Opportunities
- Define Strategic Risks
- Engage Governing Bodies
- **Activity:** Organize a FailFaire session

### Unit 9: Evaluation Tools (60 min)
- Innovation Storytelling
- Stage-Gate Assessment
- Life Cycle Analysis
- Enabling Environment Scan
- **Project:** Conduct an enabling environment scan

### Unit 10: Facilitating Dialogue & Practical Application (90 min)
- Sharing Results with Others
- Structuring the Conversation
- Facilitation Activities
- Personal Action Plan
- **Final Project:** Complete innovation action plan
